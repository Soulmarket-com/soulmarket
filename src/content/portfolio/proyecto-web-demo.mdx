## El proyecto

El cliente necesitaba una web que transmitiera la calidad y el nivel de sus proyectos de arquitectura, con un portfolio visual potente y un sistema de contacto que filtrara consultas relevantes.

## El proceso

Empezamos con una fase de descubrimiento para entender la marca, el público objetivo y la competencia. A partir de ahí diseñamos en Figma antes de pasar al desarrollo.

## Solución técnica

- **Framework**: Astro.js para máxima velocidad de carga
- **Estilos**: Tailwind CSS con sistema de diseño personalizado
- **Imágenes**: Optimización automática con Astro Image
- **SEO**: Sitemap automático, meta tags, Schema.org

## Resultados

- Tiempo de carga < 1 segundo
- Puntuación Lighthouse > 95
- 3x más contactos en los primeros 3 meses
