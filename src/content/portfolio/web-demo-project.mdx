## The project

The client needed a website that conveyed the quality and level of their architecture projects, with a powerful visual portfolio and a contact system that filtered relevant inquiries.

## The process

We started with a discovery phase to understand the brand, target audience and competition. From there we designed in Figma before moving to development.

## Technical solution

- **Framework**: Astro.js for maximum load speed
- **Styles**: Tailwind CSS with custom design system
- **Images**: Automatic optimization with Astro Image
- **SEO**: Automatic sitemap, meta tags, Schema.org

## Results

- Load time < 1 second
- Lighthouse score > 95
- 3x more contacts in the first 3 months
